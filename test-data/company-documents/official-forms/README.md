## ⚙️ 使用方式

### 上傳至 aihr 系統
1. 登入 http://localhost:3001
2. 進入文件管理
3. 直接上傳 JPG 圖片

### 系統處理流程
```
上傳 JPG → LlamaParse OCR → 文字提取 
    ↓
分塊處理 → Voyage-4-lite 向量化 → 存入 pgvector
    ↓
語義搜索引擎可查詢圖片內容
```

### 測試查詢
上傳完成後，可在聊天介面提問：

## 🔍 預期挑戰

### 可能遇到的問題
1. **圖片解析度**: 若圖片模糊，OCR 準確度會下降
2. **表格複雜度**: 多層標題或合併儲存格可能影響解析
3. **手寫文字**: 如有手寫部分，辨識難度較高
4. **圖章印鑑**: 紅色印章可能干擾文字辨識

### 優化建議
- 若辨識效果不佳，可將圖片轉為高解析度 PDF
- 使用圖片編輯工具增強對比度
- 去除不必要的背景干擾
